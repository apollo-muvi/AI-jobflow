# 接小孩通知功能規格說明

> **版本**：v1.0 | **狀態**：待開發 | **日期**：2026-06-30

---

## 📋 功能概述

家長透過 LINE Bot 發送「接小孩」通知，系統自動推播給授課老師與前台，並回覆家長確認。支援老師在考試時延遲下課時間。

---

## 🎯 核心需求

1. **家長 LINE 通知**：家長在對話中說「育成10分鐘後接」
2. **自動推播**：系統推播給授課老師與前台
3. **家長確認**：家長收到確認訊息
4. **考試延遲**：老師可回覆「考試延遲X分鐘」
5. **時間格式**：支援相對時間（10分鐘後）與絕對時間（8點）

---

## 🔄 流程設計

### 正常流程

```
家長 LINE：「育成10分鐘後接」
    ↓
【意圖識別】檢測關鍵詞（「接」、「來接」、「準備」）
    ↓
【LLM 解析】提取學生姓名、時間
    ↓
【課程查詢】查詢學生今日課程、授課老師
    ↓
【通知推播】
    ├─→ 老師 LINE：「家長【張太太】10分鐘後來接【育成】」
    ├─→ 前台 LINE：「家長【張太太】10分鐘後來接【育成】，請準備」
    └─→ 家長 LINE：「已通知老師與前台，育成準備中 ✅」
```

### 特殊情境：考試延遲

```
老師 LINE 收到通知
    ↓
老師回覆：「現在考試，延遲15分鐘」
    ↓
【系統處理】
    ├─→ 家長 LINE：「考試延遲，請15分鐘後再來 ✅」
    └─→ 前台 LINE：「育成考試延遲15分鐘」
```

## 🧠 意圖識別與解析

### 關鍵詞檢測

檢測以下關鍵詞觸發接小孩功能：
- 「接」、「來接」、「準備」
- 「育成來接」、「育成後接」

### LLM 解析（JSON 格式）

輸入範例：
```
「育成10分鐘後接」
```

輸出格式：
```json
{
  "student_name": "育成",
  "time_type": "relative",
  "time_value": 10,
  "time_unit": "minutes"
}
```

輸入範例：
```
「育成8點來接」
```

輸出格式：
```json
{
  "student_name": "育成",
  "time_type": "absolute",
  "time_value": "20:00"
}
```

### 時間解析規則

| 格式 | 範例 | 解析結果 |
|------|------|----------|
| 相對時間 | 「育成10分鐘後接」 | 當前時間 + 10 分鐘 |
| 相對時間 | 「育成5分後接」 | 當前時間 + 5 分鐘 |
| 相對時間 | 「育成半小時後接」 | 當前時間 + 30 分鐘 |
| 絕對時間 | 「育成8點來接」 | 今天 20:00 |
| 絕對時間 | 「育成5點半接」 | 今天 17:30 |
| 立即 | 「育成現在接」 | 當前時間 |

## 💾 資料庫設計

### 新增表：`pickup_notifications`

| 欄位 | 類型 | 說明 |
|------|------|------|
| id | UUID | 主鍵 |
| student_id | UUID | 學生 ID |
| parent_id | UUID | 家長 ID |
| teacher_id | UUID | 授課老師 ID |
| pickup_time | DATETIME | 預計接小孩時間 |
| actual_time | DATETIME | 實際接小孩時間 |
| status | ENUM | pending、completed、delayed |
| delay_minutes | INTEGER | 延遲分鐘數 |
| created_at | DATETIME | 建立時間 |
| updated_at | DATETIME | 更新時間 |

---

## 🔧 技術實作

| 技術 | 說明 |
|------|------|
| 意圖識別 | 關鍵詞匹配（正則） |
| 實體提取 | LLM (Z.AI GLM-5.1) + JSON 輸出 |
| 課程查詢 | SQLAlchemy 查詢 `students`、`courses`、`teachers` 表 |
| 時間解析 | Python `datetime` + `dateutil.parser` |
| LINE 推播 | LINE Messaging API (Flex Message) |

## 📝 API 設計

### POST /api/v1/pickup/notify

接收家長 LINE 通知並處理。

**請求：**
```json
{
  "line_user_id": "U1234567890",
  "message": "育成10分鐘後接"
}
```

**回應：**
```json
{
  "success": true,
  "notification_id": "uuid",
  "student_name": "育成",
  "pickup_time": "2026-06-30T20:10:00",
  "teacher_notified": true,
  "reception_notified": true
}
```

### POST /api/v1/pickup/delay

老師延遲下課時間。

**請求：**
```json
{
  "notification_id": "uuid",
  "delay_minutes": 15
}
```

**回應：**
```json
{
  "success": true,
  "new_pickup_time": "2026-06-30T20:25:00",
  "parent_notified": true,
  "reception_notified": true
}
```
